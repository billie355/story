Chapter 1: The "Luxurious" Era (Grade 12)

Back in Grade 12, I had a massive dream: I wanted to build my own social media platform from scratch. The problem? I had absolutely zero knowledge about databases, backends, or how the internet actually worked.

I was so innocent. I thought if I just typed the right HTML, magic would happen. I present to you, my masterpiece: "Luxuriuos" (yes, with that typo).

The "Magic" Login

Fun fact: You didn't need an email or password. Clicking the button just linked you straight to the next HTML page. Security level: 0.

The Sad Homepage

"No more friends... Try again later!" It was a harsh reality check that I didn't actually have a database connected. HAHAHA.

The Code Behind the Madness

I recently found the original code. Look at how many <body> tags I used! And the legendary sytle= typo.


<body><h1>Get Luxuriuos for Android or browser</h1>
</body>

The Realization...

It was funny looking back, but that innocent ambition was the spark. I realized I couldn't just "design" functionality. I needed to learn the real logic.

Chapter 2: The Glow Up (3rd Year College)

Fast forward to today. I'm now in my 3rd year of college. I traded the multiple body tags for solid fundamentals in HTML, CSS, and JavaScript.

I stopped dreaming about a social media app and actually built one. This is Catapus.

CATAPUS

Tech Stack: HTML • CSS • Vanilla JS • Firebase

No frameworks, just pure Vanilla JS controlling the DOM, hooked up to a real Firebase backend for authentication, databases, and storage.

  • ✅ Modern Dark Mode UI
  • ✅ Working Post Feed with Images
  • ✅ Real Database Integration
  • ✅ Hosted and Live
  • ✅ And yes, I gave myself a verified checkmark. 😎
Visit Catapus 🚀
